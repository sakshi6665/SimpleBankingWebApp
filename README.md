project-Simple Banking Web Application
A simple banking system web application built using Java and Spring Boot.
The application allows users to create a bank account, deposit money, withdraw money, check balance, and view transaction history.
This project is an enhanced version of a console-based banking application, upgraded to a web-based system for better user interaction and system management.

Features-
1.Create bank account with name and account number
2.Deposit money
3.Withdraw money
4.Check current balance
5.Transaction history tracking
6.Basic error handling (invalid amount, insufficient balance)
7.User-friendly web interface

Technologies Used-
1.Java
2.Spring Boot
3.Thymeleaf
4.HTML & CSS

How to Run the Application-
1.Open the project in IntelliJ IDEA
2.Run: SimpleBankingWebAppApplication.java
3.Open browser and go to: http://localhost:8080
